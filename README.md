# Startup Ecosystem Simulator

A console-based C++ project that simulates startup management, employee handling, product development, and business growth using Object-Oriented Programming and file handling.


## Features
- Create and manage multiple startups
- Hire employees
- Create and launch products
- Product development cycle system
- Revenue and budget management
- Salary deduction system
- Startup merge feature
- Automatic file saving and loading
- Persistent data storage
- Menu-driven interface


## OOP Concepts Used
- Classes and Objects
- Inheritance
- Constructors
- Encapsulation
- STL Vector
- Dynamic Data Management


## File Handling Concepts
- `ofstream`
- `ifstream`
- Automatic data persistence


## Technologies Used
- C++
- OOP
- STL Vector
- File Handling
- Console-Based UI


## Project Flow
1. Load saved startup data
2. Create or select startup
3. Hire employees and create products
4. Run business cycles
5. Launch products and generate revenue
6. Save data automatically
